GitHub Pages deployment for this project:

1. Upload the CONTENTS of this folder to the root of the public GitHub repository:
   https://github.com/slimvic7/paradise-nursery

2. In GitHub, go to Settings > Pages.

3. Select:
   Source: Deploy from a branch
   Branch: main
   Folder: /docs

4. Save and wait 2-5 minutes.

Live app:
https://slimvic7.github.io/paradise-nursery/
